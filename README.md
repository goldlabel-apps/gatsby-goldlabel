## Gatsby Template

> 👉 Public, Open Source Gatsby Template for Listingslab Goldlabel

Release 5.0.2 用于 Listingslab Goldlabel 的公共开源 Gatsby 模板


#### Install & Use

```bash
cd <working-dir>
git clone https://github.com/listingslab-goldlabel/gatsby-template
cd gatsby-template
npm install
npm run develop
```
